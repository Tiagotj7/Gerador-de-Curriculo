Sobre o Gerador de Currículos

O Gerador de Currículos foi desenvolvido para tornar a criação de currículos profissionais rápida, simples e eficiente. Com um layout limpo, é possível preencher seus dados, salvar no banco e gerar PDFs prontos para processos seletivos.

Funcionalidades Principais

Cadastro intuitivo: Formulário simples para inserir suas informações pessoais, experiência e habilidades.

Visualização do currículo: Painel em tempo real para conferir como seu currículo ficará antes de gerar o PDF.

Geração de PDF: Crie rapidamente um PDF profissional, pronto para enviar para recrutadores.

Segurança: Todos os dados são armazenados de forma segura em banco MySQL, mantendo confidencialidade.
